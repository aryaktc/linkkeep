# LinkKeep - Fullstack Project
This package contains a minimal **frontend** (React + Tailwind) and **backend** (Express + MongoDB).
Follow README files in `frontend/` and `backend/` for setup instructions.

Key features implemented:
- JWT auth (register/login)
- Profile fetch/update
- CRUD for bookmarks with search & tag filter
- Password hashing with bcrypt
- Protected backend routes with middleware
